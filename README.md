# Copy Center

> Create, store and print your documents easily with **Copy Center**

## Structure

* Documents service
* Printers service
* Gateway API

## Requirements

* docker-compose

## Build and run

1. Clone repository

2. Move to project's root

```bash
  cd cism-copy-center
```

3. Start services

```bash
  docker compose up -d --build
```

4. Check API with [http://localhost:8080/client](http://localhost:8080/client)
